

## Overview
A collection of computer vision libraries for motion analysis applications.

## Contents
This repository includes the following libraries and frameworks:

- **MMENGINE** - A foundational library for computer vision model building and training.
- **MMCV** - A foundational library for computer vision providing common utilities and components
- **MMDetection** - An object detection toolbox supporting various detection architectures
- **MMPose** - A pose estimation toolbox and benchmark for detecting human keypoints and body poses
- **Mediapipe** - Vision AI solution library.
- **hf2vad** - Video anomaly decetion code base.

## Description
This repository contains reusable libraries and utilities designed to support motion analysis tasks in computer vision projects. These libraries provide pre-trained models, training pipelines, and inference tools for pose estimation, object detection, action recognition, and video tracking.
